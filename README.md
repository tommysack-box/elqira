# Elqira

Elqira e' uno strumento web per sviluppatori pensato per comprendere il comportamento delle API.

Il progetto adotta un approccio scenario-first:

- le request vengono organizzate in scenari significativi
- le response devono essere leggibili e confrontabili
- l'obiettivo non e' solo eseguire chiamate, ma capire il flusso API

## Stack

- React
- TypeScript
- Vite

## Sviluppo locale

Requisiti:

- Node `22.12.0`

Comandi:

```bash
nvm use
npm install
npm run dev
```

Build di produzione:

```bash
npm run build
```
